## ðŸ“Š Understanding Results

**Download Speed**
- How fast you receive data
- For: streaming, browsing, downloads

**Upload Speed**
- How fast you send data
- For: video calls, cloud backups, streaming

**Ping (Latency)**
- Response time in milliseconds
- Lower is better (< 20ms = excellent)
- For: gaming, video calls

### Speed Guide

| Speed | Use Case |
|-------|----------|
| 0-25 Mbps | Email, browsing, SD video |
| 25-100 Mbps | HD video, gaming, calls |
| 100-500 Mbps | 4K video, multiple devices |
| 500+ Mbps | Heavy usage, power users |

### Ping Guide

- **0-20ms** - Excellent (gaming, calls)
- **20-50ms** - Good (smooth gaming)
- **50-100ms** - Fair (noticeable lag)
- **100+ms** - Poor (frustrating)

## ðŸ’¡ Tips

**For accurate results:**
- Close downloads/streaming
- Use Ethernet (not WiFi)
- Test multiple times
- Test different times of day
- Disable VPN for ISP test

**When to test:**
- Morning (6-9 AM) - baseline
- Evening (7-10 PM) - peak usage
- Night (11 PM-2 AM) - max speed

**Why speeds vary:**
- Network congestion
- Server load
- Other users
- Time of day
- ISP traffic management

**Best practice:** Run 5 tests, average them
